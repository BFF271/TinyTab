## 介绍

**[English](./README.md)** | 中文

一个专注于搜索的新标签页浏览器插件。

## 特性

-   🌗 深色模式
-   ⛔ 自定义搜索后缀
-   🔍 设置默认搜索引擎
-   📦 几个开箱即用的搜索引擎集成
-   🎭 通过自定义前缀快速切换搜索引擎
-   🌌 自定义任何支持 OpenSearch 的搜索引擎

## 安装

你可以在 Microsoft 应用商店中安装它，[点击这里](https://microsoftedge.microsoft.com/addons/detail/tinytab/apehhlijbbfgppmhjpmblkaocmekfmaf)! 或者通过源码构建测试版本：

1. 克隆源代码
2. 打开 Edge 或 Chrome
3. 打开扩展页
4. 开启开发者模式
5. 点击加载解压缩的扩展
6. 选择 dist 目录
