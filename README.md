## That is this

A simple new tab extension for Edge Chromium and Chrome.

## Install

-   clone 或下载源代码
-   浏览器 -> 扩展 -> 打开开发人员模式 -> 加载解压缩的扩展 -> 选择 dist 目录
-   重新打开一个浏览器标签页查看效果

## Run the build

-   `npm i`
-   `npm start`
-   `npm run build`
